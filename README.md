# omz-ezrx

A minimal, distraction-free Zsh theme focused on clarity and signal over noise.

Built for daily use — fast feedback, readable prompts, no visual clutter.

---

## Philosophy

- Information > decoration  
- Readable at a glance  
- Optimized for long terminal sessions  
- Minimal cognitive load  

If something doesn’t help you decide faster, it doesn’t belong here.

---

## Features

- Clean and readable prompt
- Git branch & status at a glance
- Clear separation between output and input
- Works well in dark terminals
- No unnecessary icons, animations, or visual noise
